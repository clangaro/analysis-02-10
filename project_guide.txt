PROJECT GUIDE — analysis_02_10
Repository: https://github.com/clangaro/analysis-02-10
Hosted app:  Streamlit Community Cloud (auto-deploys from main branch)

================================================================
STREAMLIT APP (the public-facing interactive dashboard)
================================================================

app.py                  Main entry point. Run with: streamlit run app.py
                        Contains all tabs: Hypothesis, Methods, Sleep bursts,
                        Phase x cognition (incl. correlation matrices with
                        side-by-side subgroup comparison), Effect sizes & power,
                        Diagnostics, Interpretation & implications.

app_data.py             Data loading & merging (cached). Loads CSVs, builds
                        the master table, and provides apply_filters() for
                        subsetting by sex, age, light condition, phase window.

app_plots.py            Plotly visualisation helpers — correlation heatmaps,
                        scatter plots, and other figures used by app.py.

app_stats.py            Statistics module — bootstrap CIs, Bayes factors,
                        effect sizes, FDR correction, power analysis.

requirements.txt        Python dependencies for Streamlit Cloud deployment.

STANDALONE ANALYSIS SCRIPTS
================================================================

analysis.py                     Main analysis pipeline (non-Streamlit).
analysis_overdispersion.py      Overdispersion checks for count outcomes.
barnes_pca.py                   PCA on Barnes maze variables.
circadian_behaviour_advanced.py Advanced circadian-behaviour modelling.
circadian_predicts_behaviour.py Tests whether circadian metrics predict behaviour.
correlation_matrix_analysis.py  Standalone correlation matrix computation.
exclusion_analysis.py           Sensitivity analysis for sample exclusions.
improved_barnes_analysis.py     Refined Barnes maze statistical analysis.
metrics_script.py               Reads raw PIR data, regularises, computes NPCRA
                                + cosinor circadian metrics.
normalised_pipeline.py          Normalisation pipeline and comparison.
sex_age_effects.py              Sex and age interaction effects on outcomes.
variability_analysis.py         Individual variability and residual SD analysis.
